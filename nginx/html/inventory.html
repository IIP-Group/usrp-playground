USRPs

Each row is one physical USRP. ID is a short handle used by channel mappings. Identifier is the raw UHD device-args string — e.g. serial=3485538, addr=192.168.10.2, name=myusrp, or any combination. Ports are comma-separated antenna names available on that device (e.g. TX/RX, RX2 for a B210).

Channels (TX port → RX port)

A channel is a physical wire from one USRP's TX port to another USRP's RX port. The worker uses this list when routing MIMO transmissions.
